ABOUT:

proj_r_0, also known as Rowing Sound Machine on the Apple App Store, is a web audio api as well as react native application consisting of 4 echoing channels that can be used to synthesize ambient soundscapes.

The web version you have available when clicking "SOUNDS" provides you with 4 sets of drum pads where each drum pad has either a high or low frequency equalization timbre and either a 145 bpm or 72.5 bpm feedback rate.

Additionally, each drum pad provides controls for channel feedback volume labeled "fb" and overall channel volume labeled "vol". Using these controls, you can mix the channels together to make seamless music for an endless amount of time, or drag fb controls to the left whenever you would like to gradually end a performance.

The native mobile version, available on the App Store (Here), similarly has 4 channels represented by duck and lotus icons, but has much simpler features inspired by basic drum pad user interfaces.

Both web and native proj_r_0 allow for and encourage repeated tapping/clicking of drum pad button elements in order to create fuller all encompassing soundscapes like orchestral strings.
